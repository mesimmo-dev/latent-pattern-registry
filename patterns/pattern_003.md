# Pattern 003: Threshold Triggering

## Description
A system shifts state once a specific boundary or condition is reached.

## Observed Context
Decision points, behavioral changes, alert systems, biological responses.

## Structural Notes
- Operates on boundary conditions rather than gradual change
- Often appears sudden but is preceded by accumulation
- Can be binary (on/off) or multi-state
- Threshold definition determines system sensitivity

## Indicators
- Rapid transition following incremental buildup
- Clear before/after states
- Increased system activity near boundary conditions
