# Pattern 002: Signal Compression

## Description
Information is reduced into a more compact form while preserving core meaning.

## Observed Context
Summarization systems, decision heuristics, data pipelines, communication under constraint.

## Structural Notes
- Reduces cognitive and processing load
- Introduces risk of nuance loss
- Often increases usability while decreasing fidelity
- Effective compression depends on accurate identification of signal vs noise

## Indicators
- Shortened outputs with retained structure
- Loss of qualifiers or edge cases
- Increased clarity with reduced detail
